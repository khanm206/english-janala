# 📘 English Janala

✨ *Learn English vocabulary in a structured, interactive, and beginner-friendly way.*

English Janala is a vocabulary learning web app that helps users improve their English by exploring words organized into different levels. It dynamically fetches data from an API and presents it in a clean, interactive interface.

---

## 🚀 Live Demo
🌐 https://khanm206.github.io/english-janala/

---

## 🛠️ Tech Stack

⚛️ React.js  
🟨 JavaScript (ES6+)  
📡 Axios (API handling)  
🎨 Tailwind CSS  

---

## ✨ Features

📚 **Level-based Learning System**  
- Words are organized into structured difficulty levels

🔍 **Dynamic Data Loading**  
- Fetches vocabulary data from external API in real-time

🧾 **Word Detail Modal**  
- Click any word to see meaning, pronunciation, and details

🔊 **Pronunciation Support** *(if implemented)*  
- Helps users learn correct pronunciation

🧭 **Smooth Navigation**  
- Easy switching between levels and word lists

📱 **Fully Responsive Design**  
- Works seamlessly on mobile, tablet, and desktop

⚡ **Fast Performance**  
- Optimized API calls and rendering
