# 📘 English Janala

English Janala is an interactive vocabulary learning web application designed to help users learn English words based on structured levels. It fetches data from an external API and presents words in an organized, user-friendly interface.

---

## 🚀 Live Demo
👉 https://khanm206.github.io/english-janala/

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Axios (API handling)
- Tailwind CSS

---

## ✨ Features

- 📚 Vocabulary organized by levels
- 🔍 Fetch words dynamically from API
- 🧾 Word detail view with pronunciation and meaning
- 🎯 Interactive UI with modal-based details
- 🧭 Smooth navigation between sections
- 📱 Responsive design for all devices
- ⚡ Fast and dynamic data loading

---

## 🧠 Key Learning Outcomes

This project helped strengthen:
- API integration using Axios
- Component-based UI design in React
- Dynamic routing and state management
- Handling asynchronous data
- Building structured learning interfaces
